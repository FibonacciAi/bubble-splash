# Bubble Splash!

A **bright, easy ocean adventure** for a 3–4 year old. Swim, splash with a dolphin friend, mash any key for bubbles and chaos. Original art & characters.

No fail states. Just bubbles, collecting, and happy sounds.

## Play online

**https://fibonacciai.github.io/bubble-splash/**

## Play locally

```bash
cd games/lucky-day
python3 -m http.server 8790
```

Open **http://localhost:8790**

## Controls

| Input | Action |
|-------|--------|
| **Trackpad / mouse** | Move your swimmer |
| **Any key** or **click** | Crazy splashy fun (items rain, dolphin reacts, bubbles!) |
| **🐬 SPLASH** | Call / play with the dolphin |
| **💥 BUBBLES** | Super bubble party |
| **DIVE IN** button | Start (or Space on the title) |

## For little kids

- Huge friendly art and text  
- Soft music + splashy sounds  
- Catch shells, pearls, stars, ducks  
- Cheer messages every few finds — **Splash!**  
- Nothing scary, nothing hard  

## Stack

Pure HTML / CSS / Canvas. No build step. WebAudio music + SFX.
